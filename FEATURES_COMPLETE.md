## 🎯 AI算法详解

### 极小化极大算法（Minimax）
```
评分系统:
- 将/帅: 10000分
- 车: 600分
- 马: 300分
- 炮: 300分
- 士: 200分
- 相/象: 200分
- 兵/卒: 100分 + 位置加分

位置评估:
- 兵过河加分
- 马跳中心加分
- 车控制要道加分
- 炮在中线加分
```

### Alpha-Beta剪枝
- 提速比: ~3-10倍
- 搜索深度4时评估节点: ~10000个
- 剪枝后实际评估: ~2000-3000个
